ram_concept.shrinkage_loading_layer module¶

class ShrinkageLoadingLayer(uid, model)¶

Bases: LoadingLayer

ShrinkageLoadingLayer represents a named layer that contains ShrinkageAreaLoad, and their results.

The LoadingType and LoadingAnalysisType of ShrinkageLoadingLayers cannot be changed.
