class DefaultTendonSegment(uid, model)¶

Bases: TendonSegment

DefaultTendonSegment represents a TendonSegment that stores the default values for future TendonSegments that are created.

An Exception will be raised if the location, node_1 or node_2 properties of DefaultTendonSegment are accessed.

There is only 1 and always 1 DefaultTendonSegment in the Model. It is accessed through CadManager.default_tendon_segment

class TendonSegment(uid, model)¶

Bases: CadEntity

TendonSegment is a tendon segment between profile points.

TendonSegments are always located on a TendonLayer.

  • model (Model) –

elevations_along_segment(fractional_lengths)¶

Determines the elevations at all the given fractional distances along the TendonSegment. 0.0 is the fractional length for end 1 and 1.0 is the fractional length at end 2. Values are only guaranteed to be correct after a Call All.
