ram_concept.structure_layer module¶

class StructureLayer(uid, model)¶

Bases: CadLayer

StructureLayer represents the “mesh input” layer in the CAD system.

The structure drawn on the StructureLayer is meshed into the finite elements on the ElementLayer. The StructureLayer is accessible through CadManager.structure_layer.
