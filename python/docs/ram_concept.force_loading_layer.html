ram_concept.force_loading_layer module¶

class ForceLoadingLayer(uid, model)¶

Bases: LoadingLayer

ForceLoadingLayer represents a named layer that contains force-defining loads, and their results.

Some ForceLoadingLayers (balance, self-dead and hyperstatic) are read-only. Modification operations attempted on those layers will result in exceptions.
