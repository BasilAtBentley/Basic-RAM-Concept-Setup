ram_concept.line_support module¶

class DefaultLineSupport(uid, model)¶

Bases: LineSupport

DefaultLineSupport represents a LineSupport that stores the default values for future LineSupport that are created.

An Exception will be raised if the location property of DefaultLineSupport is accessed.

There is only 1 and always 1 DefaultLineSupport in the Model. It is accessed through CadManager.default_line_support

class LineSupport(uid, model)¶

Bases: RigidSupport

LineSupport is a logical (on/off) support at a line segment that provides support to slabs.

The r-axis for the support is defined as along the location segment from start to end. The s-axis is horizontal and perpendicular to the location segment.

LineSupports are always located on the StructureLayer (“Mesh Input” layer).
