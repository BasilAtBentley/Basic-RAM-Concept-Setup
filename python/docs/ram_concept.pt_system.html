ram_concept.pt_system module¶

class PTSystem(uid, model)¶

Bases: Data

PTSystem represents a post-tensioning system.

To create a new PTSystem, use PTSystems.add_pt_system. To see all PTSystems that exist in a Model, use PTSystems.pt_systems.

Parameters:

  • uid (int) –

  • model (Model) –

delete()¶

Delete the PTSystem mix from the Model. The last PTSystem in a Model cannot be deleted.

Return type:

None
