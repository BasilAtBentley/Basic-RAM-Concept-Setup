ram_concept.slab_opening module¶

class DefaultSlabOpening(uid, model)¶

Bases: SlabOpening

DefaultSlabOpening represents a SlabOpening that stores the default values for future SlabOpenings that are created.

An Exception will be raised if the location property of DefaultSlabOpening is accessed.

There is only 1 and always 1 DefaultSlabOpening in the Model. It is accessed through CadManager.default_slab_opening

  • model (Model) –
