ram_concept.bracket_parser module¶

class BracketParser(parse_string)¶

Bases: object

This class provides utilities for parsing strings that have square bracket [] delimited tokens.

Nested bracket strings ([A][[B1][B2]]) are supported. The class operates on one depth at a time. Typically, you will just want to call BracketParser.Parse(string).
