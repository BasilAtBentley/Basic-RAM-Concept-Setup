ram_concept.element_layer module¶

class ElementLayer(uid, model)¶

Bases: CadLayer

ElementLayer represents the meshed structure, containing elements, supports and springs.

The API does not provide any significant mechanisms the change the ElementLayer contents as they are intended to be generated through meshing. The ElementLayer is accessible through CadManager.element_layer.
