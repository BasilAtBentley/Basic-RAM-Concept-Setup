ram_concept.strand_material module¶

class StrandMaterial(uid, model)¶

Bases: Data

StrandMaterial represents a strand in a post-tensioning system.

To create a new StrandMaterial, use StrandMaterials.add_strand_material. To see all StrandMaterials that exist in a Model, use StrandMaterials.strand_materials.

  • model (Model) –
