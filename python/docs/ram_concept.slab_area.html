ram_concept.slab_area module¶

class DefaultSlabArea(uid, model)¶

Bases: SlabArea

DefaultSlabArea represents a SlabArea that stores the default values for future SlabAreas that are created.

An Exception will be raised if the location property of DefaultSlabArea is accessed.

There is only 1 and always 1 DefaultSlabArea in the Model. It is accessed through CadManager.default_slab_area

  • model (Model) –
