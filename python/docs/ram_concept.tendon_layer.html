ram_concept.tendon_layer module¶

class TendonLayer(uid, model)¶

Bases: CadLayer

TendonLayer represents a manual or generated tendon layer in the CAD system.

The TendonLayer is accessible through CadManager.tendon_layer and CadManager.tendon_layers.

Parameters:

  • uid (int) –

  • model (Model) –

add_jack(location)¶

Add a Jack at the given location, copying properties from CadManager.default_jack.

Note that the location will be snapped to the nearest 0.1mm and a TendonNode may be created if none exist at that location.

Parameters:

location (Point2D) –

Return type:

Jack

add_tendon_segment(location)¶

Add a TendonSegment at the given location, copying properties from CadManager.default_tendon_segment.

Note that the location will be snapped to the nearest 0.1mm and 1 or 2 TendonNode may be created if none exist at that the end points.
