ram_concept.beam module¶

class Beam(uid, model)¶

Bases: ConcreteSpanningMember

Beam represents a beam along a line segment in the CAD system.

Beams are always located on the StructureLayer (“Mesh Input” layer).

class DefaultBeam(uid, model)¶

Bases: Beam

DefaultBeam represents a Beam that stores the default values for future Beams that are created.

An Exception will be raised if the location property of DefaultBeam is accessed.

There is only 1 and always 1 DefaultBeam in the Model. It is accessed through CadManager.default_beam
