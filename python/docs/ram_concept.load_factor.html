ram_concept.load_factor module¶

class LoadFactor(uid, model)¶

Bases: Data

LoadFactor represents the load factors with which a LoadingLayer is included in a LoadComboLayer.

LoadFactors are created and deleted automatically as LoadComboLayers and LoadingLayers are created and deleted.

If the relevant LoadComboLayer.summing_type is LoadComboSummingType.LATERAL_GROUP and this LoadFactor refers to a lateral loading, then this LoadFactor and its values are ignored.

If the relevant LoadComboLayer.analysis_type is LoadComboAnalysisType.NONLINEAR then the alternate_load_factor is ignored.

LoadFactors are accessed through LoadComboLayer.load_factor.
