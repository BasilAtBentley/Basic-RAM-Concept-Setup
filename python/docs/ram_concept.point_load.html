ram_concept.point_load module¶

class DefaultPointLoad(uid, model)¶

Bases: PointLoad

DefaultPointLoad represents a PointLoad that stores the default values for future PointLoads that are created.

An Exception will be raised if the location property of DefaultPointLoad is accessed.

There is only 1 and always 1 DefaultPointLoad in the Model. It is accessed through CadManager.default_point_load

Parameters:

  • uid (int) –

  • model (Model) –
